Setup & Integration Guide

1. Azure Entra ID App Setup

To interface with Microsoft Graph APIs, register an application in your Azure Portal:

1
Create Registration Create a new App Registration in the Azure Entra ID portal.
2
Generate Secrets Go to Certificates & Secrets and generate a Client Secret value.
3
Add Application Permissions Under API Permissions, add the following Application permissions (not Delegated):
Calendars.ReadWrite Mail.Send OnlineMeetings.ReadWrite.All CallTranscript.Read.All CallRecording.Read.All
4
Grant Admin Consent Click the "Grant Admin Consent for [Organization]" button. (Crucial!)

2. Testing Webhooks Locally

Graph subscription updates require a public HTTPS callback endpoint:

1
Launch Tunnel Run ngrok in terminal to expose port 3000: ngrok http 3000.
2
Copy URL Copy the generated secure URL (e.g. https://...ngrok-free.app).
3
Register Subscriptions Save it under Webhook Base URL, then click below to register subscriptions.
