Setup & Integration Instructions

1. Azure Entra ID App Setup

To connect to Microsoft Graph, you must register an application in Azure:

  • Create an App Registration in Azure Entra ID.
  • Create a Client Secret and paste it into the credentials form.
  • Add the following Application Permissions:
    • Calendars.ReadWrite
    • CallTranscript.Read.All
    • CallRecording.Read.All
  • Click "Grant Admin Consent" in Azure. (Essential!)

2. Testing Graph Webhooks Locally

Microsoft requires an HTTPS endpoint to send subscriptions alerts:

  1. Install ngrok globally or download it.
  2. Run ngrok http 3000.
  3. Copy the https://...ngrok-free.app URL.
  4. Paste it into the Webhook Base URL field on this page and save.
  5. Use the Register Webhooks button below to subscribe.
